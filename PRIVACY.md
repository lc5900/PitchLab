# Privacy

PitchLab is designed for local pitch practice.

## Microphone

On Android, PitchLab asks for microphone permission so it can detect pitch in real time. Audio frames are processed locally and are not saved as recordings.

## Stored Data

PitchLab stores practice summaries locally, including:

- Practice mode.
- Target note when target mode is used.
- Duration.
- Average pitch deviation.
- Stability percentage.
- Pass/fail result.

PitchLab does not intentionally store raw audio.

## Network

The app does not currently require network access for pitch detection or practice history.

## Language Settings

The selected UI language may be stored locally so the app can restore it on the next launch.
